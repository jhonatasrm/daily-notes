# Daily Notes

App for iOS to daily notes, stored in CoreData with the possibility of:

* Add note
* Remove Note
* Edit note
* Date and hour of the note




# Notas Diárias

App para iOS de notas diárias, armazenados no CoreData de forma persistente com a possibilidade de:

* Adicionar nota
* Remover nota
* Editar notar
* Data e Hora da nota
